# HR_Analytics_using_PowerBI_Tableau_SQL_Excel
 Interactive dashboards for HR analytics using Power BI, Tableau, and SQL. Providing  insights into employee attrition, job satisfaction, department trends, and demographics. Designed to assist organizations in optimizing workforce management and employee retention strategies.












# HR Analytics Dashboard

Developed by [Monalisa Akter](https://github.com/MonalisaAkter), this project presents an interactive HR Analytics Dashboard designed to provide comprehensive insights into employee data, focusing on attrition rates, job satisfaction, and demographic trends.

## Features

- **Employee Count**: Displays the total number of employees.
- **Attrition Analysis**: Examines attrition counts and rates, including breakdowns by gender, department, and education field.
- **Active Employees**: Shows the number of currently active employees.
- **Average Age**: Calculates the average age of employees.
- **Job Satisfaction**: Analyzes job satisfaction ratings across different job roles.

## Tools and Technologies

- **Data Visualization**: Power BI, Tableau
- **Data Analysis**: SQL
- **Programming**: Python

## Usage

1. **Data Setup**: The `hrdata` table is created and populated with employee data.
2. **SQL Queries**: Various SQL queries are executed to extract insights, such as employee counts, attrition rates, and job satisfaction metrics.
3. **Visualization**: The results are visualized using Power BI and Tableau to create interactive dashboards.
